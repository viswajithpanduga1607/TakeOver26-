What's Next for Verdikt AI

We're just getting started. Here's what's on our roadmap to make autonomous workflows even smarter.

📊
▾

Manager Dashboard

A dedicated interface for managers to view all requests, decisions, and reasoning in one place.

The dashboard will pull live data from Google Sheets, presenting it in a clean, searchable table with filters by employee, request type, and decision. Managers can spot-check AI reasoning and override decisions when needed, creating a human-in-the-loop safety layer.

🧠
▾

Persistent Agent Memory

The AI will remember past decisions and patterns per employee, improving consistency over time.

By implementing window buffer memory in n8n, the agent will maintain context across requests — recognizing patterns like frequent leave requests, escalating repeat complaints, and ensuring consistent treatment of similar cases across the organization.

🌐
▾

Multi-language Support

Extend request submission and AI evaluation to regional languages.

Leveraging Gemini's multilingual capabilities, employees will be able to submit requests in their native language. The AI will understand, evaluate, and respond appropriately while logging standardized English entries for managerial review.

⚙️
▾

Custom Escalation Rules

Let organizations define their own approval thresholds and escalation logic.

Admins will configure rules like "auto-approve leave under 3 days" or "always escalate expenses over ₹10,000." The AI agent will respect these organizational policies while still applying contextual reasoning to edge cases.

📈
▾

Analytics Dashboard

Visual insights into approval trends, rejection reasons, and team request patterns.

Interactive charts and trend lines will surface actionable insights — which departments file the most complaints, peak leave request seasons, common rejection reasons, and AI decision accuracy over time. Data-driven HR management at its finest.
